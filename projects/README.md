# Projects

Completed deliverables with before/after comparisons. Each project demonstrates end-to-end AI-augmented work.

## What Makes a Good Portfolio Project

1. **Clear problem** - Business context is obvious
2. **Visible process** - Shows iteration, not just output
3. **Measurable impact** - Quantified where possible
4. **Honest assessment** - Includes what AI couldn't do

## Project Categories

### Strategic
Long-term planning, roadmaps, market analysis

### Tactical
Sprint-level work, feature specs, user stories

### Operational
Process improvements, documentation, training
